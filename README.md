# КодИмпульс | Киберпанк терминал

![Preview](https://cdn.poehali.dev/templates/cyberpunk-terminal-ru/preview-ru.jpg)

Интерактивный лендинг в стиле киберпанк с матричным дождём, командной строкой и неоновыми эффектами. Погрузитесь в атмосферу цифрового андеграунда.

## Особенности

- Интерактивный терминал с командами
- Анимированный матричный дождь на фоне
- Неоновое свечение и эффекты
- Тёмная киберпанк-тема
- Адаптивный дизайн
- TypeScript + React + Vite
- Tailwind CSS + shadcn/ui

## Быстрый старт

```bash
npm install
npm run dev
```

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Lucide Icons
